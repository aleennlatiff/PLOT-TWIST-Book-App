# PLOT-TWIST Book App

Hi! This is my midterm project for the Basic Programming 2 class. My professor is Deniz Dahman. 

I made a mobile application to discover new books and save them. 

## How I made this project:
* I used Flutter and Dart to write the code.
* I used a girly theme because I wanted the app to look nice and friendly. I used pink and baby blue colors.
* I used a `TextField` to make a search bar. This way, you can type and search for books easily.
* I used `ChoiceChip` to make category buttons like Fiction or Sci-Fi.
* I used a `GridView` to show the books on the home screen. I used 3 items in one row so the book images look small and fit the screen well.
* I used the `Hero` widget to make a cool animation. When you click on a book, the picture flies to the next page.
* I used a simple `List` to save the books. When you click the heart button on the details page, I used `setState` to update the screen and put the book in the "My Library" page.

## Notes:
In this project, I used a local list for the saved books. Because of this, if you close the app completely, your saved books will disappear. I didn't use a real database yet. But I learned a lot about how to design an app and make UI in Flutter!
